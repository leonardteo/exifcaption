# exifcaption

1. Install exiftool. `brew install exiftool`
2. Take this script, install it somewhere e.g. `~/bin` make it executable: `chmod +x ~/bin/exiftool`. Add `~/bin` directory to your shell path.
3. Run it on a folder to generate captions e.g. `exiftool *`
